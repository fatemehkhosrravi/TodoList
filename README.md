Todo List Application
## Project Description
A simple and practical task management application written in C++. With this program, you can manage your daily tasks by setting priorities and deadlines.

## Features
- ** Add new task** with title, description, creation date, deadline, and priority
- ** Show all tasks** with complete details
- ** Show completed tasks**
- ** Show uncompleted tasks**
- ** Show delayed tasks** (tasks past their deadline)
- ** Change task status** (done/undone)
- ** Remove task**
- ** Sort by priority** (Low, Medium, High)
- ** Sort by deadline**

##  Project Structure
- `main.cpp` - Main file and program menu
- `Task.h` and `Task.cpp` - Base task class
- `Todo.h` and `Todo.cpp` - Derived class from Task
- `TodoList.h` and `TodoList.cpp` - Task list management class

##  How to Run

Follow these steps to run the project:

1. Download the `project_programing(1).zip` file
2. Extract the file
3. Open the project files in your preferred development environment
4. Run the codes

##  Usage Guide
After running, the following menu is displayed:
1. Add new task
2. Show all tasks
3. Show completed tasks
4. Show uncompleted tasks
5. Show delayed tasks
6. Change task status
7. Remove task
8. Sort by priority
9. Sort by deadline
10. Exit

**Note:** Priorities: 1=Low, 2=Medium, 3=High
